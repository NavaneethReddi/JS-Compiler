# JS-Compiler

A simple online JavaScript compiler and runner built with HTML, CSS, and JavaScript. Features a CodeMirror editor for writing code and a console output panel for results.

## Features

- Syntax-highlighted JavaScript editor
- Real-time console output
- Support for ES6+ features
- Responsive design

## Deployment

This project is ready for deployment on Vercel as a static site.

### Steps to Deploy on Vercel:

1. Push this repository to GitHub.
2. Go to [Vercel](https://vercel.com) and sign in.
3. Click "New Project" and import your GitHub repository.
4. Vercel will automatically detect it as a static site and deploy it.
5. Your app will be live at the provided URL.

No additional configuration is needed since the entry file is `index.html`.
